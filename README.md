# GraphTheory-MatchingProject
Repository for the Matching Schools and Students Project (ENSEEIHT - First Year Graph Theory Course).


To setup the project's environment, execute setup.sh.
To run the project, execute run.sh.

A Python test file is available in the app/utils folder. To run it, execute "python3 -m app.utils.test.py" from the base folder ("GraphTheory-MatchingProject").
